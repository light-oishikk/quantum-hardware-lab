Quantum Hardware Interactive Lab

Course: A Primer on Quantum Computers — IIIT Dharwad

This tool provides interactive, browser-based simulations for five core quantum hardware platforms studied in the course. All computations are performed in real-time using first-principles physics models.

Computation Engine:

• Jacobi Eigenvalue Solver for Hamiltonian diagonalization (Transmon, NV Center)
• Lindblad Dynamics analytical solutions for decoherence evolution
• Mathieu Stability pixel-level stability region computation for ion traps
• Fock State Algebra beam splitter transformations for photonic circuits

No pre-computed data — every plot is generated from the underlying equations when you click Compute. All matrix diagonalizations happen live in the browser.

Zero external dependencies • Pure HTML5/CSS3/JavaScript • Canvas API rendering • ~50KB total

Developed by Oishik Kar • IIIT Dharwad • 2026

Physics Reference

Lindblad Master Equation:
d rho/dt = -i[H,rho] + Sum_k (L_k rho L_k^dag - 1/2 {L_k^dag L_k, rho})
Amplitude Damping (T1):
rho_11(t) = rho_11(0) exp(-t/T1), rho_01(t) = rho_01(0) exp(-t/2T1)
Pure Dephasing:
1/T2 = 1/(2T1) + 1/T_phi. Off-diag: rho_01(t) = rho_01(0) exp(-t/T2)
Constraint:
T2 ≤ 2T1 (always enforced in physical systems)
