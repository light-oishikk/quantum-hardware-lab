# Quantum Hardware Interactive Lab

**IIIT Dharwad** | Primer on Quantum Computers

An interactive, browser-based quantum hardware simulation suite covering all five units of the course curriculum. All computations use first-principles physics models — no pre-computed data.

## Live Demo

**[Launch the Lab →](https://light-oishikk.github.io/quantum-hardware-lab/)**

## Modules

| Module | Unit | Physics |
|--------|------|---------|
| **Decoherence Lab** | I | Real-time Bloch sphere with Lindblad dynamics, T₁/T₂ decay, density matrix evolution |
| **Transmon Designer** | II | Cooper Pair Box Hamiltonian diagonalization via Jacobi eigenvalue solver |
| **Photonic Circuit** | III | Hong-Ou-Mandel interference, beam splitter transformations, Mach-Zehnder interferometry |
| **Ion Trap Sim** | IV | Mathieu stability diagram (pixel-rendered), secular frequencies, Rabi oscillations |
| **NV Center Lab** | V | Spin-1 Hamiltonian, Zeeman splitting, ODMR spectrum simulation |

## Computation Engine

- **Jacobi Eigenvalue Solver** — Real symmetric matrix diagonalization for Hamiltonian spectra
- **Lindblad Master Equation** — Analytical solutions for amplitude damping, phase damping, and depolarizing channels
- **Mathieu Stability** — Pixel-level computation of stable/unstable regions in the (a,q) parameter space
- **Fock State Algebra** — Beam splitter unitary transformations with photon distinguishability

## Technical Details

- **Zero dependencies** — Pure HTML5, CSS3, and Vanilla JavaScript
- **Single file** — Everything runs from one HTML file (~50KB)
- **Real-time rendering** — HTML5 Canvas API for all visualizations
- **No server required** — Open `index.html` directly in any modern browser
